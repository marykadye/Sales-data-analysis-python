
# 📊 Sales Data Analysis Project (Python)

## 🔍 Overview
This project is a simple data analysis case study built using Python. It demonstrates how to load, process, and analyze sales data to generate meaningful business insights.

The goal is to simulate real-world data analysis tasks such as understanding revenue, product performance, and sales trends.

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Jupyter Notebook

---

## 📁 Dataset Description
The dataset contains basic sales transaction data with the following columns:

- Date: Transaction date
- Product: Item sold
- Quantity: Number of units sold
- Price: Price per unit

---

## 📊 Key Analysis Performed
The project performs the following analysis:

- Calculation of total revenue per transaction
- Overall total sales calculation
- Identification of best-selling product
- Sales breakdown by product

---

## 💡 Key Insights
- Revenue is calculated using Quantity × Price
- Product performance can be compared using total revenue
- Some products contribute significantly more to total sales than others
